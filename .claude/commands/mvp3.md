THESIS MVP 3단계를 시작한다.

목표: Free 플랜 대시보드 — 국면 신호 + 기본 자산배분 UI

현재 상태: $ARGUMENTS

구현할 화면:
1. 메인 대시보드 — 국면 신호 카드 (리스크온/중립/리스크오프) + 감정 언어 설명
2. 자산배분 — 슬라이더 3개 (투자기간/리스크허용도/투자금액) → 주식·현금·안전자산 비율 자동 계산
3. 기본 지표 3개 — VIX, 원달러, 외국인 수급 (수치 대신 신호등 색상)

설계 철학 필수 적용:
- 결론 먼저 (DXY 수치 직접 노출 금지)
- 감정 언어 사용 (-20% 빠지면 잠 못 잔다)
- 과신 방지 문구 UI에 포함
- 다크모드 기본

제약:
- shadcn/ui + Recharts만 사용
- 모바일 우선 반응형
- Skeleton 로딩 처리 필수
- Pro 전용 기능은 blur + 잠금 아이콘으로 미리보기

먼저 계획만 제시하고 구현하지 말 것.
