THESIS MVP 1단계를 시작한다.

목표: Next.js 14 + TypeScript + Tailwind + shadcn/ui 세팅 + Supabase Auth (이메일/Google 로그인)

현재 상태: $ARGUMENTS

제약 조건:
- App Router 사용 (Pages Router 금지)
- TypeScript strict 모드
- shadcn/ui 컴포넌트만 사용 (커스텀 UI 라이브러리 추가 금지)
- 환경변수는 모두 .env.local에 (하드코딩 금지)
- Supabase RLS 처음부터 활성화

출력물:
1. 구현 계획 (단계 목록) → 내 승인 후 진행
2. 승인 후 파일 생성

먼저 계획만 제시하고 구현하지 말 것.
