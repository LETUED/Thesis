# 14 · portfolio 빈상태 screener 동선 (portfolio-empty-screener-link)

- **작업**: portfolio 빈상태 안내가 /lab(beta 조립)만 가리켜, 더 단순한 /screener를 보조 경로로 추가.
- **브랜치**: `cycle_2` · **세대·회차**: cycle_2 / 14 (무한모드 14회차, 자율)

## ★ 사용자-가치
**입문자가 관심종목을 처음 담을 때, beta 조립 분석보다 단순한 종목 검색(screener)도 안내받아 부담을 줄인다.**

## 조사 (cy1, Explore) — 안전 범위 정직 판정
- **#1 채택(소·안전)**: portfolio 빈상태에 /screener 보조 링크. 동작 페이지 간 안내라 "준비중" 정직성 무관, 종목 미제시로 매수권유 철학 충돌 없음.
- **#2 배제(medium·정직성)**: dashboard ④ "준비중" 카드 링크 — `/screener`는 실제 단일 조회기인데 "종목 스크리너"라 부름 → 라벨 간극 노출은 부정직. 카피 재정의 선행 필요.
- **#3 배제(medium+·철학 회색)**: 국면 검색어 시드 — 국면 연동 시 시점 권유 위험. 철학 가드(정적 섹터·비연동·추천문구금지) 정의 선행 필요.

## 진행 (cy5) — 변경 파일 1
- `frontend/components/personalization/MyWatchSection.tsx` — 빈상태 action에 `/screener`("기업 검색하기") 버튼을 기존 `/lab`과 병렬 추가. description도 "종목 검색이나 조립 분석"으로.

**검증**: 타입체크 클린 · 빌드 그린.

## 리뷰 (cy6) — trivial 동선·위험 아님 → 자가점검
- 회귀 0(빈상태 action JSX, 기존 lab 버튼 보존) / 정확성(screener 동작·인증게이트 동일이라 막다른 리다이렉트 없음) / 철학(종목 미제시) / 정직성(동작 페이지 안내). 외부 오라클 그린.

## 상태
- ✅ **그린** — 타입체크·빌드. portfolio→screener 동선 보강.
- **⚠️ 자율 가치 한계 도달**: 14 cy1이 정직 판정 — **안전하게 자율로 할 수 있는 작은 입문자 가치는 #1로 소진**. 남은 5단계 심화(#2 정직성 재정의·#3 철학 가드·필터 스크리너 구현)는 모두 **사람 스펙/방향이 선행돼야 하는 medium+**.
- **A/B 메모(핵심)**: 14회차 무드리프트. v2 하네스가 **"안전한 작은 가치 소진 → medium+는 사람 의도 필요"를 정직 표면화**(억지 갭 금지). cycle_1(15회차 표류)과 결정적 대조 — cycle_2는 14회차에 자율 한계를 정직히 인식.
